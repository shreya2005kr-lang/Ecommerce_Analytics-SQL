# E-Commerce Analytics using SQL

This project focuses on analyzing e-commerce customer and transaction data using SQL and SQLite.

## Objective
To understand customer distribution, revenue performance and key business metrics such as average order value and customer segments.

## Tools Used
- SQL
- SQLite
- DB Browser for SQLite

## Dataset
The project uses multiple relational tables including:
- customers
- transactions

## Analysis Performed
- Country wise customer distribution
- Country wise revenue analysis
- Revenue by loyalty tier
- Average Order Value (AOV) by country
- Top customers by total revenue
- Acquisition channel performance

## Files
- ecommerce.db  
  (SQLite database containing all tables)

- ecommerce_analysis.sql  
  (SQL queries used for analysis)

## Key Insights
- Revenue contribution varies significantly by country.
- Bronze and Silver segments contribute the highest overall revenue.
- Gold customers generate the highest average revenue per customer.
- Organic and Paid Search channels drive the highest revenue.
